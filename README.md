# Monokai For IntelliJ IDEA 11.

## Supported Languages:

### Now: 

- Scala
- Javascript
- CSS
- SASS
- LESS
- HTML
- XML
- Closure Templates

### Later:

- MOAR Scala
- MOAR Java
- Python
- PHP
- Django/Jinja Templates
- Twig Templates
- Localization


### Laterer:

- Ruby
- HAML
- CoffeeScript
- SQL

## Installation

### OS X

Place the .xml in ~/Library/Preferences/IdeaIC11/colors/ and restart IDEA.

### Windows
Place the .xml in %HOMEPATH%.IntelliJIdea11\config\colors and restart IDEA.
